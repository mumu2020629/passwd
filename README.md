## 简介



该工具是使用python3开发的针对元素进行组合生成密码字典的工具，可以快速针对收集到的敏感信息快速进行排列组合。



## 使用说明


直接下载releases版本即可

**使用python启动，命令如下：**

```
python3 demo.py
```
